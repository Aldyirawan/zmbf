### Cara Install Tools
````
pkg update && pkg upgrade
pkg install git python
rm -rf zmbf 
git clone https://github.com/Fall-Xavier/zmbf
````
### Cara Install Bahan Tools
````
cd zmbf
pip install -r asset/requirements.txt
python build.py
````
### Cara Menjalankan Tools
````
python run.py
````
